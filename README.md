Smart City Management System (v2.0)A comprehensive C++ application designed to simulate and manage various urban sectors using advanced Data Structures and Algorithms. This system utilizes a Graph-based architecture to represent the city's infrastructure, connecting transportation, commercial, and public sectors. OverviewThe Smart City Management System provides an interactive CLI to manage a digital twin of a city. It integrates multiple sectors into a single unified graph where locations are nodes and paths are edges. Key Features & SectorsPopulation Sector: Manage citizen records, housing data, and perform demographic analytics (Age distribution, Gender ratios, etc.).Education Sector: Maintain school hierarchies (Organograms), student/faculty registration, and school rankings.Commercial Sector: Detailed mall management including floor-by-floor shop directories and product category searching.Medical Sector: Centralized health facility management and emergency service coordination.Transport Sector:Graph Theory: Uses Dijkstra’s Algorithm to find the shortest path between bus stops.Simulation: Real-time bus journey simulation and passenger drop-off mechanics.Public Facilities: Management of mosques, parks, and other essential city services. Project StructureFileDescriptionmaaaain.cppThe entry point of the application. Handles the main menu and input validation.SmartCity.hContains the core Graph Data Structure that links all city sectors.Population.hLogic for citizen management and CSV-based data persistence.Education.hManages school data, departments, and educational hierarchies.Commercial.hHandles mall registration, shop inventory, and nearest mall calculations.Medical.hInterface for the city's medical facilities and healthcare data.Transport.hLogic for bus stops, route connectivity, and Dijkstra-based pathfinding.PublicFacilities.hManagement system for community spaces like parks and mosques. Technical ImplementationData Structures UsedGraphs: Used to represent the city map (Nodes = Locations, Edges = Roads/Routes).Trees/Hierarchies: Used in the Education sector for school organograms.Hash Maps / Tables: For efficient searching of citizens by CNIC or products by category.Linked Lists / Vectors: For managing dynamic collections of buses and facilities.AlgorithmsDijkstra's Algorithm: For calculating the most efficient transport routes.Search Algorithms: Optimized string searching for facility and product location.Data Validation: Custom robust validation for integer, double, and string inputs to prevent buffer overflows or crashes.
